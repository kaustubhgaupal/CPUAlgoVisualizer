# CPUSchedulerAI üöÄ

A powerful, interactive **CPU Scheduling Simulator** combined with an **AI-powered Teaching Assistant**.

This project helps students and developers visualize how operating systems schedule processes (like FCFS, Round Robin, SJF) and provides a real-time **AI Chatbot** (powered by Google Gemini) to explain concepts, answer questions, and analyze scheduling logic.

## ‚ú® Key Features

### üìä Scheduling Simulator
- **6 Algorithms Supported:**
  - First-Come, First-Served (FCFS)
  - Shortest Job First (Non-Preemptive)
  - Shortest Job First (Preemptive)
  - Priority Scheduling (Non-Preemptive)
  - Priority Scheduling (Preemptive)
  - Round Robin (with customizable Time Quantum)
- **Interactive Gantt Chart:** Visual timeline showing exactly when each process runs.
- **Real-time Metrics:** Automatically calculates:
  - Completion Time
  - Turnaround Time
  - Waiting Time
  - Averages
- **Dynamic Process Management:** Add, remove, and configure arrival times, burst times, and priorities.

### ü§ñ AI Tutor (New!)
- **Integrated Chatbot:** Built with Google Gemini 2.0 Flash.
- **Context Aware:** Ask definitions, comparisons, or logic behind the algorithms.
- **Rich Text Support:** Responses feature bold text, lists, and formatted explanations for better readability.
